## 📜주요 기능 소개
- 관리자를 위한 백 오피스 기능
  - 강의 관리
  - 회원 관리
  - 게이머 관리

- 난이도 별, 게이머 별 강의 조회 기능

- 출석 체크를 통한 포인트 획득 기능

- 원하는 강의를 포인트를 사용하여 해금하는 기능

- 사용자의 의견을 표출하는 건의 및 자유 게시판 기능

- [프로젝트  API 명세서 보러가기](https://www.notion.so/dokuny/API-5e9845cf6f4f412194aaf99b61b0a615)
